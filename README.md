# ISIL - Programación JavaScript y TypeScript

Este paquete contiene documentación base en formato Markdown para presentar, adjuntar y explicar el proyecto integrador del curso.

## Proyecto propuesto
**Plataforma de Gestión de Cursos e Inscripciones**

## Propósito del paquete
Organizacion del trabajo del curso como un proyecto progresivo en sesiones y por capas:

- diseño base en HTML + CSS
- fundamentos con JavaScript moderno y TypeScript
- panel administrativo en Angular
- API REST con Node.js + Express + MongoDB
- portal del estudiante en React
- landing pública en Next.js
- despliegue y seguridad al cierre del ciclo
