Sobre este curso

Este curso te llevará desde los fundamentos de React hasta el desarrollo de aplicaciones completas con manejo de estado, enrutamiento y consumo de APIs REST. Aprenderás con proyectos reales, siguiendo buenas prácticas del ecosistema frontend moderno.

Al finalizar, serás capaz de construir SPAs profesionales utilizando React 18, integrar librerías del ecosistema y gestionar proyectos con arquitectura escalable lista para producción.

¿Qué aprenderás?

✅ Componentes funcionales y JSX
✅ useState, useEffect, useContext
✅ React Router v6
✅ Consumo de APIs con Axios
✅ Redux Toolkit para estado global
✅ Testing con Vitest + Testing Library
✅ Deploy con Vite + Vercel
✅ Proyecto final: SPA completa
