# 02 - Historias de usuario

## Roles principales
- Administrador
- Docente
- Estudiante

## Historias de usuario del administrador

### HU-01
Como administrador, quiero registrar cursos para mantener actualizada la oferta académica.

### HU-02
Como administrador, quiero editar la información de un curso para corregir horarios, modalidad o docente.

### HU-03
Como administrador, quiero desactivar cursos para evitar inscripciones fuera de vigencia.

### HU-04
Como administrador, quiero visualizar cuántos cursos, docentes y estudiantes existen para tener una vista general del sistema.

### HU-05
Como administrador, quiero gestionar docentes para asignarlos correctamente a los cursos.

### HU-06
Como administrador, quiero gestionar estudiantes para validar su estado dentro del sistema.

## Historias de usuario del estudiante

### HU-07
Como estudiante, quiero registrarme en la plataforma para acceder a los cursos.

### HU-08
Como estudiante, quiero iniciar sesión para consultar mis datos e inscripciones.

### HU-09
Como estudiante, quiero ver el catálogo de cursos para identificar opciones disponibles.

### HU-10
Como estudiante, quiero revisar el detalle de un curso para evaluar si me conviene inscribirme.

### HU-11
Como estudiante, quiero inscribirme en un curso para reservar mi participación.

### HU-12
Como estudiante, quiero revisar mis inscripciones para conocer mi estado actual.

### HU-13
Como estudiante, quiero consultar mi perfil para validar mis datos personales.

## Historias de usuario del docente

### HU-14
Como docente, quiero visualizar los cursos que tengo asignados para revisar mi carga.

### HU-15
Como docente, quiero consultar información básica de los cursos para preparar el dictado.
